# RobotArm Al5D Interface & Driver

An interface and driver for the Lynxmotion AL5D written in C ++ and ROS (Robot Operating System). Here the code is divided and modularly designed into Embedded System Layers such as: Application Layer, System Software Layers, Device Drivers & Hardware Layer
